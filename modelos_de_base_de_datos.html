MODELOS DE BASE DE DATOS

Los modelos de bases de datos son estructuras conceptuales que determinan cómo se organizan, almacenan y gestionan los datos en un sistema de gestión de bases de datos (DBMS). Estos modelos definen las reglas y estándares para la organización de datos y las relaciones entre ellos. Los modelos 3 modelos mas utilizados actualmente son:

• MODELO DE RELACION

• MODELO NOSQL

• MODELO OREINTADO A OBJETOS
