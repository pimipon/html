MODELO RELACION

El modelo relacional organiza los datos en tablas (relaciones) que están compuestas por filas (tuplas) y columnas (atributos). Cada tabla tiene una clave primaria que identifica de manera única cada fila. Las relaciones entre tablas se definen mediante claves foráneas.

Ventajas:
•  Estructura clara : La organización en tablas con filas y columnas es fácil de entender y utilizar.

• Consistencia de datos : Mediante el uso de restricciones y reglas de integridad, se mantiene la consistencia de los datos.

• SQL : El uso del Lenguaje de Consulta Estructurado (SQL) permite realizar consultas complejas de manera eficiente.
